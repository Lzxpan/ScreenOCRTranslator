# ScreenOCRTranslator
使用快速啟動鍵啟動，框選要翻譯的句子，可以將翻譯的結果顯示在框選處。<br>

使用說明<br>
1.設定Gemini API key。<br>
2.選取目標語系。<br>
3.選取OCR或AI辨識。<br>
4.程式縮到最小。<br>
5.按下Q鍵+滑鼠左鍵啟動畫面框選功能。<br>
6.框選需要翻譯的句子。<br>
7.翻譯結果會顯示在框選處。<br>
